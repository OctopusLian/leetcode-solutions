# 海康威视2017校园秋季招聘技术支持工程师面试经验  

面试单位：杭州海康威视数字技术股份有限公司  
面试岗位：技术支持工程师  
面试时间：2017年10月22日下午  
面试地点：本校  
面试过程：

```
    先询问了我一些基本的家庭状况，算是缓和下面试的气氛。
    紧接着开始问我假如去一些比较艰苦的地方，有没有问题？我思索了一下，说“应该没问题”。（注意：此处可以把“应该”去掉的）。
    让我用C语言写了一个函数，实现某种功能，感觉蛮轻松的。
    询问了我获过的奖项、参与的比赛。
    问我平时爱看哪些书，技术类和非技术类的，我说了我近两年看过的技术类书籍和文学类书籍。
    问我平时有没有其他兴趣爱好，我说打篮球，乒乓球；紧接着面试官问我去过哪些地方，我说新疆、玉树、还有去年单人旅行时去过的地方。
    问我有没有兴趣去杭州总部工作，我说有。
    面试结束。
```

 
面试总结：  

```
    面试一定要带纸质简历！！！（因为我是第一次面试，只在网申通道提交了我的个人电子简历，所以纸质简历没有打印）
    对于面试官问你是否愿意去艰苦岗位工作时，一定要给予肯定的答复（是/否），而不是一个模糊的答案，这样会让面试官无法和你沟通。
    比赛奖项、项目经验是加分项，可适当向面试官提一下。（其实你不说，他也会问的）
    面试时不要太紧张。
    最后补充一下，面试前一定要事先了解清楚这个岗位是干嘛的。比如我其实一直想应聘算法或研发岗位，但面试官告诉我这个岗位就是经常出差，给客户调试设备。当我知道后，我就不想面了，所以直接给面试官说其实我是想着先准备考研的。后来面试被刷，我也并不感到沮丧。
```

 
写在最后：  

```
     昨天听的海康威视的宣讲会，感觉很棒。但由于要准备考研，所以只好放弃。但晚上的时候负责人给我说可以先通过网申有个保障，后面无法完成笔试和面试的时候给我说一声就好，不影响考研。所以我就花了10分钟完成了网申（简历是以前在智联招聘上写好的，所以直接导入就好了）。

    第一次面试，没有穿正装，没有准备纸质简历，不过面试官并没有说什么。而且我是最后一个面试的，因为前面要排队，所以我就想着等大家都面完了我再去。（其实后来想想这样很不好，面试官会以为我对他们的公司不感兴趣，或者期望度不高，导致面试官对你的印象不好，这是我面试完后才顿悟的，所以下次面试我坚决不能犯这种低级错误。）

   面试结束后，面试官给我的建议是先好好准备考研，职位可以先给你留着，并让我记一下他的联系方式（预示着面试已经泡汤了）。他想让我投总部的技术岗，但可能会由于我的竞争力不强而导致求职失败。

   确实！其实仔细想想，鱼和熊掌不可兼得，现在最主要的还是要考研，因为这是一次提升自身竞争力的机会。不过此次面试所积累的经验，可以为下次更好的完成面试做好准备。加油！
```
